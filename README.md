# Employee Productivity Prediction Web App

This is a Flask-based machine learning web application that predicts employee productivity based on input features.

## 🚀 Features
- User-friendly web interface (HTML, CSS)
- Machine learning model integration (pickle)
- Real-time prediction output
- End-to-end data flow from UI to backend

## 🛠️ Technologies Used
- Python
- Flask
- HTML, CSS
- NumPy, Pandas, Scikit-learn

## 📂 Project Structure
- app.py → Flask backend
- model.py → ML model training
- gwp.pkl → Trained model
- templates/ → HTML files
