About This Project
This project focuses on predicting employee productivity using Machine Learning techniques. The goal is to help organizations understand and improve workforce efficiency.
We analyze various factors such as department, working hours, incentives, idle time, and more to predict how productive an employee will be.
Multiple machine learning models were trained and compared, including:
• Linear Regression
• Random Forest
• XGBoost (Best Performing Model)
Based on performance metrics like MSE, MAE, and R² Score, the best model was selected and deployed.
This web application allows users to input employee data and instantly get productivity predictions.
